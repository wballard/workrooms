The Solution

Video, Voice, & Text - Right in Your Browser

Web Browser FTW

Workrooms is an always on, virtual presence work system based around audio and video collaboration that is just as easy as dropping by your desk.

Workrooms extends your browser for collaboration. No need to remember to use or install another app.

Calls come in while you are working, and can be simply left in a browser tab. No more window shuffling.

Peer-To-Peer Privacy

Workrooms runs calls over WebRTC, the next generation of video/audio/screen conference technology. The magic is that it connects you directly, the server doesn't ever touch your call.

No Metadata

The Workrooms Server, throws call data away as soon as you disconnect. And only keeps call data in memory.
